# Makeup Game - Интерактивная игра-макияж


## Описание проекта

Интерактивная игра для мобильных устройств, в которой игрок может наносить макияж на лицо персонажа с помощью перетаскивания. Игрок управляет рукой, берет косметические средства и наносит их в зону лица.

### Основные механики

- **Drag-управление рукой** - перетаскивание руки с предметом к лицу
- **Крем** - убирает прыщи с лица
- **Тени** - наносятся на веки (9 цветов на выбор)
- **Помада** - красит губы (6 цветов на выбор)
- **Спонжик** - мгновенно стирает весь макияж

- ---

## Игровой процесс

1. Нажмите на предмет (крем, цвет теней или помаду)
2. Рука автоматически берет предмет
3. Перетащите руку к лицу
4. Отпустите в зоне лица для нанесения
5. При отпускании вне зоны - предмет возвращается без эффекта

###  Основные классы

### Главный контроллер руки

HandController

Главный контроллер руки. Управляет взятием предметов, перетаскиванием и нанесением макияжа.

```csharp
public void TakeCream()     // Взять крем
public void PickColor()     // Взять кисточку с цветом  
public void TakeLipstick()  // Взять помаду
public bool IsHandBusy()    // Проверка, занята ли рука
```

FaceController
Управляет внешностью персонажа.
```csharp
public void RemoveAcne()          // Убрать прыщи
public void ApplyShadow()         // Нанести тени
public void ApplyLipstick()       // Нанести помаду
public void ClearAllMakeup()      // Стереть весь макияж
```
DragHandler
Обрабатывает перетаскивание руки с ограничением по границам экрана.
